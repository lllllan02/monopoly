# Monopoly Project

这是一个参考多个开源项目设计和开发的联机大富翁游戏。

## 快速启动

你可以使用根目录下的 `Makefile` 快速启动项目：

- `make install`: 安装服务端和客户端的所有依赖
- `make server`: 启动后端服务
- `make client`: 启动前端服务
- `make dev`: 同时启动后端和前端
