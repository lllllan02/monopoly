# 大富翁（Monopoly）技术栈说明

为了实现本地运行并支持局域网联机，本项目建议采用以下成熟、易部署的技术方案。

## 1. 核心技术选型

### 前端 (Client & Admin)
*   **框架:** React + Vite (提供极致的开发热更新体验)
*   **状态管理:** Zustand 或 Redux Toolkit (管理游戏全局状态)
*   **实时通信:** Socket.io-client (处理与服务端的双向实时同步)
*   **UI 组件库:** Ant Design (用于后台管理) + Tailwind CSS (用于游戏界面)
*   **游戏渲染:** (可选) 若需 3D 效果可使用 Three.js (React Three Fiber)

### 后端 (Game Server & Admin API)
*   **后端运行环境:** Node.js (推荐 v18+)
*   **实时引擎:** Socket.io (处理房间和实时游戏逻辑)
*   **数据库 (一切从简):** JSON 文件存储 (使用 **Lowdb** 或直接通过 `fs` 模块读写)
*   **数据结构:** 地块、地图、卡片等配置均存储在本地 `/data/*.json` 文件中

## 2. 局域网联机方案

为了让局域网内的其他用户能够加入游戏，需要进行以下配置：

### 2.1 服务端配置
*   **监听地址:** 服务端需要监听 `0.0.0.0` 而非 `127.0.0.1`，以允许外部 IP 访问。
*   **静态资源:** 前端编译后的代码由后端或简单的静态服务器托管。

### 2.2 访问方式
1.  **获取主机 IP:** 在主机（运行服务器的电脑）终端运行 `ipconfig` (Windows) 或 `ifconfig/ip addr` (Mac/Linux) 获取局域网 IP（例如 `192.168.1.5`）。
2.  **防火墙开通:** 确保主机防火墙允许所使用的端口（如 `3000` 或 `8080`）的入站流量。
3.  **客户端连接:** 其他玩家在浏览器输入 `http://192.168.1.5:3000` 即可进入游戏。

## 3. 开发环境要求
*   **Node.js:** v18.x 或更高
*   **包管理工具:** pnpm (推荐) 或 npm/yarn
*   **数据库:** 本地安装 MySQL 或使用 Docker 快速启动

## 4. 快速启动流程 (预期)
1.  `git clone ...`
2.  `pnpm install` (安装依赖)
3.  配置 `.env` 文件（设置数据库连接和端口）
4.  `pnpm dev` (启动开发服务器)
5.  获取本机局域网 IP 并告知好友
